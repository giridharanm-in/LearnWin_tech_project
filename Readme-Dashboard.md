# 🎓 LearnWin Tech Academy – Student Analytics Dashboard

An end-to-end **Student Analytics Dashboard** built using **MySQL** and **Power BI** to monitor student registrations, attendance, course performance, instructors, and revenue.

---

## 📌 Project Overview

This project helps an educational institute analyze its operations through interactive dashboards. It provides insights into student enrollment, attendance trends, course popularity, instructor performance, and revenue generation.

### **Objectives**
- Track student registrations
- Analyze attendance performance
- Monitor course enrollments
- Evaluate instructor performance
- Analyze revenue trends
- Support business decision-making

The database is connected to Power BI to create interactive reports and business KPIs.
